# notes_organiser
## To install:
### 1. Install dependencies
#### On arch-based distros:
    sudo pacman -S git gcc qt5-base
### 2. Clone repository
    git clone https://github.com/kleist0202/notes_organiser.git 
### 3. Enter the following commands
    cd notes_organiser  
    qmake
    make

## To run:<br />
    ./notes_organiser
